How to Use

Include the font in your CSS:

@font-face {
    font-family: 'NewAED';
    src: url('path/to/Newaed-Regular.ttf') format('truetype');
    font-weight: normal;
    font-style: normal;
}

Apply the font to specific elements:

.aed-symbol {
    font-family: 'NewAED', sans-serif;
}

Use in HTML:

<span class="aed-symbol">AED</span> 199.99
